read_connections_spreadsheet
============================


.. py:function:: singlecell_input_mapper.singlecell_input_mapper.reader.read_connections_spreadsheet(fname)

   Reads a spreadsheet with connection probabilities between cell types

   :Parameters: **fname** (*str*) -- The name of the file to be read

   :returns: A dictionary with the following structure: {EXC: {celltype: {norm: value, ...}, ...}, INH: {...}}
   :rtype: dict

   .. rubric:: Example
      :class: example

   >>> connectionsSpreadsheet = read\_connections\_spreadsheet('connections.csv')
   >>> connectionsSpreadsheet
   {'EXC': {
       celltype\_1: {
           'SOMA\_LENGTH': 0.1, 'SOMA\_AREA': 0.2, ...
           },
       celltype\_2: {...},
       },
   'INH': {...}
   }
