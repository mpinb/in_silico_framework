write_scalar_field
==================


.. py:function:: singlecell_input_mapper.singlecell_input_mapper.writer.write_scalar_field(fname=None, scalarField=None)

   These can be visualized in AMIRA, or converted to VTK using :py:meth:`~visualize.vtk.convert\_amira\_lattice\_to\_vtk`
   for visualization in VTK-compatible renderers.

   :Parameters: * **fname** (*str*) -- Name of the output file
                * **scalarField** (:py:class:`~singlecell\_input\_mapper.singlecell\_input\_mapper.scalar\_field.ScalarField`) -- Scalar field to be written to disk

   :returns: None. Writes the results to disk.
