.. py:class:: singlecell_input_mapper.singlecell_input_mapper.cell.Cell
   
   Cell object for mapping synapses onto a morphology.

   This is a leightweight dataclass specialized for use with :py:mod:`singlecell_input_mapper.singlecell_input_mapper.synapse_mapper`.

   .. seealso::

      This is not the same class as :py:class:`single_cell_parser.cell.Cell`.
      Contrary to :py:class:`single_cell_parser.cell.Cell`, this class does not provide any biophysical details,
      simulation parameters or biophysical details.

   :Attributes:

   .. attribute:: id

      Unique identifier for the cell.

      :type: str

   .. attribute:: soma

      The soma section.

      :type: :py:class:`~singlecell_input_mapper.singlecell_input_mapper.cell.PySection2`

   .. attribute:: structures

      Dictionary mapping section labels (e.g. "Soma", "Dendrite" ...) to a list of corresponding sections.

      :type: dict

   .. attribute:: sections

      List of all sections.

      :type: list

   .. attribute:: boundingBox

      Bounding box around the cell.

      :type: tuple

   .. attribute:: synapses

      Dictionary mapping presynaptic cell types to a list of :py:class:`~singlecell_input_mapper.singlecell_input_mapper.cell.Synapse` objects.

      :type: dict

   :Methods:
