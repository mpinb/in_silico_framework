filter_short_ISIs
=================


.. py:function:: spike_analysis.core.filter_short_ISIs(t, tdelta=0.0)

   Filters out any events that occur in an interval shorter than tdelta.

   :Parameters: * **t** (*list*) -- A list containing timepoints of events.
                * **tdelta** (*float, optional*) -- Minimal interval between events. Defaults to 0.

   :returns: A list containing filtered spike times.
   :rtype: list

   .. rubric:: Example
      :class: example

   >>> filter\_short\_ISIs([1,2,3,4,5], tdelta=1.5)
   [1, 3, 5]
   >>> filter\_short\_ISIs([1,2,3,4,5], tdelta=2)
   [1, 3, 5]
