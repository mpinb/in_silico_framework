get_period_label_by_time
========================


.. py:function:: spike_analysis.core.get_period_label_by_time(periods, t)

   Classifies timepoint based on whether it falls within a period.

   :Parameters: * **periods** (*dict*) -- dictionary containing period label as key and (period\_start, period\_end) as value
                * **t** (*float*) -- timepoint to classify

   :returns: label of period in which t is in. 'undefined' if t is in no defined period
   :rtype: str
