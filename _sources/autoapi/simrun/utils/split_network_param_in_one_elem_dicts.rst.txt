split_network_param_in_one_elem_dicts
=====================================


.. py:function:: simrun.utils.split_network_param_in_one_elem_dicts(dict_)

   Split a network parameter dictionary into a list of dictionaries.

   This method is used to split a network parameter dictionary into a list of dictionaries, each containing only one element
   for each key in the original dictionary.

   :Parameters: **dict\_** (dict | :py:class:`~single\_cell\_parser.parameters.ParameterSet`) -- The network parameter dictionary.

   :returns: A list of dictionaries, each containing only one element of the original dictionary.
   :rtype: list
