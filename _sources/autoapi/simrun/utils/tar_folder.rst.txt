tar_folder
==========


.. py:function:: simrun.utils.tar_folder(source_dir, delete_folder=True)

   Compress a folder to ``.tar`` format.

   :Parameters: * **source\_dir** (*str*) -- The path to the folder to compress.
                * **delete\_folder** (*bool*) -- If ``True``, the original folder will be deleted after compression.

   :returns: Compresses the folder in place.
   :rtype: None

   :raises RuntimeError: If the compression command fails
