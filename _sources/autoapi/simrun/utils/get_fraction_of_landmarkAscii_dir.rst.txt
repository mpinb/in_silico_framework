get_fraction_of_landmarkAscii_dir
=================================


.. py:function:: simrun.utils.get_fraction_of_landmarkAscii_dir(frac, basedir=None)

   Sample landmarks from all landmarkAscii files in a directory.

   This method loads all landmarkAscii files in a directory and returns a DataFrame containing the sampled landmarks and the cell type.

   :Parameters: * **frac** (*float*) -- Fraction of landmarks to sample.
                * **basedir** (*str*) -- The path to the directory containing the landmarkAscii files.

   :returns: A pandas DataFrame containing the sampled landmarks and the cell type.
   :rtype: pd.DataFrame

   .. seealso:: :py:meth:`~simrun.utils.get\_fraction\_of\_landmarkAscii` to sample landmarks from a single file.
