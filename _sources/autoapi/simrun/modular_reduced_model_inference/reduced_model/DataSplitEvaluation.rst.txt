.. py:class:: simrun.modular_reduced_model_inference.reduced_model.DataSplitEvaluation(Rm)
   
   Split data in training-test sets.

   This class provides an interface to split input data in separate train-test sets,
   and to evaluate the reduced model performance scores corresponding to the splits.

   :Parameters: **Rm** (:py:class:`Rm`) -- Reduced model. Set after running :py:meth:`setup`

   :Attributes:

   .. attribute:: Rm

      Reduced model. Set after running :py:meth:`setup`

      :type: :py:class:`Rm`

   .. attribute:: splits

      Dictionary of splits

      :type: dict

   .. attribute:: solvers

      List of solvers

      :type: list

   .. attribute:: optimizer_results

      List of optimization results

      :type: list

   .. attribute:: optimizer_results_keys

      List of optimization results keys

      :type: list

   .. attribute:: scores

      List of scores

      :type: list

   .. attribute:: scores_keys

      :type: list
