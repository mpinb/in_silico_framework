.. title:

fig2np
======


.. py:function:: visualize._figure_array_converter.fig2np(fig)

   Converts fig-object to pixels as numpy arrays.

   This method converts figure objects to numpy arrays.
   These numpy arrays are RGB pixel matrices.

   .. seealso:: http://stackoverflow.com/questions/7821518/matplotlib-save-plot-to-numpy-array

   :Parameters: **fig** (*matplotlib.pyplot.Figure*) -- The figure object to convert to a numpy array.

   :returns: The numpy array of the figure.
   :rtype: numpy.ndarray
