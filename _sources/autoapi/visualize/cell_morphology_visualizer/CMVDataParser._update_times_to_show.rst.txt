CMVDataParser._update_times_to_show
===================================

.. py:method:: visualize.cell_morphology_visualizer.CMVDataParser._update_times_to_show(t_start=None, t_stop=None, t_step=None)

   Checks if the specified time range equals the previously defined one. If not, updates the time range.
   If all arguments are None, does nothing. Useful for defining default time range

   :Parameters: * **t_start** (*float*) -- start time
                * **t_stop** (*float*) -- end time
                * **t_step** (*float*) -- time interval

   :returns: None. Updates :paramref:`times_to_show`.
