CMVDataParser._get_voltages_at_timepoint
========================================

.. py:method:: visualize.cell_morphology_visualizer.CMVDataParser._get_voltages_at_timepoint(time_point)

   Retrieves the coltage along the whole cell morphology from cell object at a particular time point.
   Each voltage is defined per section in the morphology.
   Note that the array of data per section each time starts with the last point of its parent section.

   Fetches this voltage from the original cell object.

   :Parameters: **time_point** (*float|int*) -- time point from which we want to gather the voltage

   :returns: list of voltage points for each morphological point in the cell at :paramref:`time_point`.
   :rtype: voltage_point (list)
