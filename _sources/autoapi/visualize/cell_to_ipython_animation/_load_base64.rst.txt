_load_base64
============


.. py:function:: visualize.cell_to_ipython_animation._load_base64(filename, extension=&#39;png&#39;)

   Load a file and encode it in base64.

   This is used to load images and embed them in an html file, such as Jupyter notebooks.

   :Parameters: * **filename** (*str*) -- path to file
                * **extension** (*str*) -- the file extension. Default: 'png'

   :returns: base64 encoded string
   :rtype: str

   .. seealso:: https://github.com/jakevdp/JSAnimation/blob/master/JSAnimation/html\_writer.py
