get_index
=========


.. py:function:: biophysics_fitting.exploration_from_seedpoint.RW_analysis.get_index(dataframe, channel)

   Compute the depolarization or hyperpolarization index.

   This index is how much the given channel contributed to depolarization (or hyperpolarization),
   relative to the total depolarization (or hyperpolarization) current.

   :Parameters: * **dataframe** (*pd.DataFrame*) -- the dataframe containing the biophysical parameters
                * **channel** (*str*) -- the channel for which to compute the index

   :returns: the depolarization/hyperpolarization index for the given channel
   :rtype: pd.Series
