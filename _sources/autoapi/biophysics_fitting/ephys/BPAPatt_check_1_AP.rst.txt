BPAPatt_check_1_AP
==================


.. py:function:: biophysics_fitting.ephys.BPAPatt_check_1_AP(t, v_soma, thresh=None, stim_onset=None)

   Checks if there is exactly one action potential in the somatic voltage trace.

   Args:
   - t (array): time array [ms]
   - v\_soma (array): somatic voltage trace [mV]
   - thresh (float): spike threshold [mV]
   - stim\_onset (float): time of stimulus onset [ms]

   Returns:
   - bool: True if there is exactly one action potential, False otherwise
