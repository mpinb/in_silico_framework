BAC_ISI_check_repolarization
============================


.. py:function:: biophysics_fitting.ephys.BAC_ISI_check_repolarization(t, v, stim_end=None, repolarization=None)

   Checks if the membrane potential has repolarized to a certain value after a stimulus for a BAC stimulus.

   :Parameters: * **t** (*numpy.ndarray*) -- Array of time values.
                * **v** (*numpy.ndarray*) -- Array of membrane potential values.
                * **stim\_end** (*float, optional*) -- Time at which the stimulus ends. Defaults to None.
                * **repolarization** (*float, optional*) -- Value to which the membrane potential should repolarize. Defaults to None.

   :returns: True if the membrane potential has repolarized to the specified value, False otherwise.
   :rtype: bool
