.. title:

AP_height
=========


.. py:function:: biophysics_fitting.ephys.AP_height(t, v, thresh=None)

   Computes the amplitude of each action potential (AP) in a voltage trace.

   :Parameters: * **t** (*numpy.ndarray*) -- Array of time values.
                * **v** (*numpy.ndarray*) -- Array of voltage values.
                * **thresh** (*float, optional*) -- AP threshold voltage. If None, uses the default threshold of find\_crossing().

   :returns: Array of AP amplitudes.
   :rtype: numpy.ndarray
