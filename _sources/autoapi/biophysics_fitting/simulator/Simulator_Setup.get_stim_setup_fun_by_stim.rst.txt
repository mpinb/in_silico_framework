Simulator_Setup.get_stim_setup_fun_by_stim
==========================================

.. py:method:: biophysics_fitting.simulator.Simulator_Setup.get_stim_setup_fun_by_stim(stim)

   Get the stimulus setup function by stimulus name.

   Stimulus setup functions are functions that set up the stimulus.
   They are saved under the name ``stimulus_name.setup``, and accessible
   under :paramref:`stim_setup_funs`.

   :Parameters: **stim** (*str*) -- The stimulus name, e.g. ``'stim_1'``, ``'bAP'``, ``'StepOne'``.

   :returns: The setup function for the stimulus.
   :rtype: Callable
