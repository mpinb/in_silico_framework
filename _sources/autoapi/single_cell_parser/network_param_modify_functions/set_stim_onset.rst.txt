.. title:

set_stim_onset
==============


.. py:function:: single_cell_parser.network_param_modify_functions.set_stim_onset(n, onset=None)

   Changes the offset when pointcells get activated

   :Parameters: * **n** (:py:class:`~single\_cell\_parser.parameters.ParameterSet` | dict) -- The :ref:`network\_parameters\_format`.
                * **onset** (*float*) -- The onset time in milliseconds.

   .. rubric:: Example
      :class: example

   >>> celltype = 'L6cc\_C2'  # layer 6 cortico-cortical cells in column C2
   >>> n.network[celltype].celltype.pointcell.offset
   0
   >>> set\_stim\_onset(n, 100)
   >>> n.network[celltype].celltype.pointcell.offset
   100
