_get_locking_config
===================


.. py:function:: data_base.distributed_lock._get_locking_config()

   Get the locking configuration from the environment variable ``ISF\_DISTRIBUTED\_LOCK\_CONFIG``.

   This method allows for a lazy evaluation of the locking configuration,
   and is used to set the global variable ``\_CONFIG``.
