create_metadata
===============


.. py:function:: data_base.db_initializers.load_simrun_general.metadata_utils.create_metadata(db)

   Generate metadata out of a pd.Series containing the sim\_trial\_index.

   Expands the sim\_trial\_index to a pandas Series containing the path, trial number,
   and filename of the voltage traces file. After running this method, the database
   contains:

   - ``sim\_trial\_index`` A pandas series containing the sim\_trial\_index.
   - ``simresult\_path`` Path to the simulation results folder.

   :Parameters: **db** (:py:class:`~data\_base.isf\_data\_base.isf\_data\_base.ISFDataBase`) -- The target database that should contain the parsed simulation results.

   :returns: A pandas dataframe containing the metadata.
   :rtype: pd.DataFrame
