_get_dumper
===========


.. py:function:: data_base.db_initializers.load_simrun_general.utils._get_dumper(value, categorized=False)

   Infer the best dumper for a dataframe.

   Infers the correct parquet dumper for either a pandas or dask dataframe.

   :Parameters: **value** (*pd.DataFrame or dd.DataFrame*) -- Dataframe to infer the dumper for.

   :returns: Dumper module to use for the dataframe.
   :rtype: module

   :raises NotImplementedError: If the dataframe is not a pandas or dask dataframe.
