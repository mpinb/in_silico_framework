colorize_str
============


.. py:function:: data_base.utils.colorize_str(key, bcolor)

   Colorize a string with a terminal color.

   Simply concatenates the string with escape sequences that colroize the text in a terminal.

   :Parameters: * **key** (*str*) -- the string
                * **bcolor** (*str*) -- the color

   :returns: the colorized string
   :rtype: str

   .. seealso:: :py:class:`data\_base.utils.bcolors` For color escape sequences.
