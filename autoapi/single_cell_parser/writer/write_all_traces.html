    single_cell_parser ❭ writer ❭ write_all_traces

    write_all_traces¶

    single_cell_parser.writer.write_all_traces(fname, t, vTraces)¶

    Write out a list of voltage traces.

    Parameters:¶

    • fname (str) – The name of the file to write to.

    • t (list) – The time points of the voltage traces.

    • vTraces (list) – A list of voltage traces.

    Returns:¶

    None. Writes out the voltage traces to fname.

    Example

    >>> t = [0.0, 0.1, 0.2]
>>> vTraces = [[-65.0, -64.9, -64.8], [-70.0, -69.9, -69.8]]
>>> write\_all\_traces('voltage\_traces.dat', t, vTraces)
